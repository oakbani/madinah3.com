یہ بلاگ Astro میں لکھا گیا ہے۔ ہر پوسٹ ایک سادہ Markdown فائل ہے جسے Astro خود بخود ایک صفحے میں بدل دیتا ہے۔

## فہرست کی مثال

- پہلی شے
- دوسری شے
- تیسری شے

> اقتباس بھی RTL میں درست نظر آتا ہے۔

```js
// کوڈ ابھی بائیں سے دائیں رہتا ہے — یہی مناسب ہے۔
const salam = "السلام علیکم";
console.log(salam);
```
