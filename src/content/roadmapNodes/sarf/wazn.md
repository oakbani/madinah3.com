---
title: Patterns (الوزن)
---

A **wazn** (literally "weight") is a template that root letters drop into. The
classical placeholder root is **ف‑ع‑ل**, so patterns are written using ف, ع, ل.

Examples on root **ك‑ت‑ب**:

| Pattern | Word | Meaning |
| --- | --- | --- |
| فَعَلَ | كَتَبَ | he wrote |
| فَاعِل | كَاتِب | writer |
| مَفْعُول | مَكْتُوب | written |
| فِعَال | كِتَاب | book |

Learn the patterns and you can decode thousands of words.
